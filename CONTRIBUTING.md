# Contribuindo para o Synapse-Admin

Fala Dev! Quer ajudar a melhorar o Synapse-Admin? Que top.
Pra manter tudo organizado, segue algumas regars básicas:

## Como contribuir
1. **Fork o repositório** e clone no seu Pc.
2. Crie uma branch pro seu feature/bugfix:
    ```bash
    git checkout -b feature/ome-da-sua-feature
    ```
3. Faça as alterações no código e testa tudo antes de mandar.
4. Commit suas mudanças com mensages claras:
    ```bash
    git commit -m "Adicionei recurso X / Corrige bug Y"
    ```
5. Faça psuh para sua branch:
    ```bash
    git push origin feature/nome-da-sua-feature
    ```
6. Abra um pull Request(PR) explicando suas mudanças.

# Regras Importantes 

* Código limpo, bem comentado e organizado.
* Seguir padrões do Laravel e PSR.
* Teste tudo antes de enviar PR.
* Nenhum código malicioso ou que quebre funcionalidades existentes.

# Bugs e Sugestões 
* Abra um issue explicando o problema ou a sugestão;
* Tente ser claro e detalhado, screenshots ajudam muito.

Obrigado por contribuit.
