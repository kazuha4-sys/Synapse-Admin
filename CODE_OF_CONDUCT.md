# Conduta para o Synapse-Admin

Nosso objetivo é criar um ambiente saudável e amigvel para todo mundo que participa do Synapse-Admin.


## Regras Básicas
- Seja **respeitoso** com todos.  
- Não serão tolerados **assédio, discurso de ódio ou comentários ofensivos**.  
- Seja paciente com iniciantes.  
- Ajude de forma construtiva, críticas devem ser **educadas e construtivas**.

## Como Reportar Problemas
Se alguém violar este código de conduta, abra uma **issue privada** ou entre em contato com os mantenedores.  
Toda denúncia será tratada de forma confidencial.

## Consequências
Quem violar este código de conduta pode ter suas contribuições removidas e, se necessário, ser **bloqueado do repositório**.

Obrigado por manter o Synapse-Admin seguro e amigável!
